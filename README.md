# Launch KPI Portal v1

업로드된 `[GGT] O X M Launching KPI 비교.xlsx` 파일의 주요 시트를 기반으로 만든 정적 웹 프로토타입입니다.

## 포함 화면
- Dashboard: 런칭 성과 Overview
- Game Comparison: 게임별 1day / 10days / 15days / 30days / 60days KPI 비교
- Ranking: 기간/지표별 랭킹
- Pre-registration: 사전예약 성과 비교
- Admin Design: 향후 엑셀 업로드/DB 저장 구조 설계

## 실행 방법
`index.html` 파일을 브라우저에서 열면 됩니다.
